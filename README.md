# demo-dunia

Repositorio que contiene la demo de la aplicación `demo-dunia`.

## Estructura

- `api/chat.js` - Lógica de backend para el chat.
- `public/index.html` - Interfaz web de la demo.

## Uso

1. Abre `public/index.html` en un navegador o sirve la carpeta `public` con un servidor estático.
2. Asegúrate de que `api/chat.js` esté disponible desde el backend si tu demo lo requiere.
